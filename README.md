1.
**โจทย์**  
ให้อ่านข้อมูลอุณหภูมิ (หน่วยองศาเซลเซียส) แล้วคำนวณหาค่าองศาฟาเรนไฮต์และเคลวิน  
โดยใช้สูตร:  
- F = (9/5) * C + 32  
- K = C + 273.15  
**ข้อมูลเข้า**  
- หนึ่งบรรทัด ประกอบด้วยค่าอุณหภูมิหน่วยองศาเซลเซียสเป็นจำนวนจริง
**ข้อมูลออก**  
- หนึ่งบรรทัด ประกอบด้วยตัวเลขจำนวนจริง 2 จำนวน คั่นด้วยเว้นวรรค  
  จำนวนแรกคือค่าอุณหภูมิหน่วยองศาฟาเรนไฮต์  
  จำนวนที่สองคือค่าอุณหภูมิหน่วยเคลวิน
c = float(input("องศาเซลเซียส: "))
f = 9/5 * c + 32
k = c + 273.15
print(f"องศาฟาเรนไฮต์คือ: {f} องศาเคลวินคือ: {k}")

2.
**โจทย์**  
จงเขียนโปรแกรมเพื่อคำนวณหาพื้นที่สามเหลี่ยม
**ข้อมูลนำเข้า**  
- บรรทัดที่ 1: รับค่าฐาน เป็นเลขจำนวนจริง  
- บรรทัดที่ 2: รับค่าความสูง เป็นเลขจำนวนจริง
**ข้อมูลส่งออก**  
- บรรทัดเดียว แสดงพื้นที่สามเหลี่ยม เป็นเลขจำนวนจริง
- base = float(input("ค่าของฐานคือ: "))
height = float(input("ค่าของความสูงคือ: "))
area = base * height * 1/2
print(f"พื้นที่สามเหลี่ยมคือ: {area:f}")
